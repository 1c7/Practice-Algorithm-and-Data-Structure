##  一句话总结归并排序(Merge sort)
不断往下切, 切到剩下一个的时候就开始合并了  
比较两个数组的第一个元素  
把小的那一个放在排序结果数组里(假设你要的排序结果是从小到大)  
用到了递归 recursive. 这种算法是分而治之的 divide-and-conquer
